# vui

> It's not another component library, but an exploration about what is the right way to build a component library

### 项目结构
```
vui
 |-- build
 |-- config
 |-- src
      |-- doc // 文档
      |-- lib // 组件库
           |-- styles       // CSS 组件
           |-- directives   // 自定义指令
           |-- filters      // 自定义过滤器
           |-- utils        // 纯 js 组件 
           |-- widgets      // vue 组件
      |-- index.js // 导出模块的出口文件
 |-- test // 单元测试
 |-- package.json
 |-- README.md
```

### 步骤

- 开发
- 编写测试
- 运行测试
- 更新文档并上传至 github page
- commit 变更信息
- 合并 Git
